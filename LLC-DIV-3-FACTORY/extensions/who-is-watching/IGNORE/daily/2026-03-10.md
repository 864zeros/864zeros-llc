**Date:** 2026-03-10
**Session Duration:** Extended session

## Executive Summary

Transformed "Who Is Watching" from a static URL-pattern-based tracker detector into a dynamic behavioral fingerprinting engine. The extension now detects trackers by **what they DO**, not just where requests go. This addresses the core problem: only catching ~5 vendors on major sites because URL patterns miss first-party proxies, CNAME cloaking, GTM-injected tags, and custom analytics endpoints.

## What Was Built Today

### 1. Behavioral Fingerprinting Engine (`injected/hook.js`)

**Complete rewrite** - 1000+ lines of behavioral detection code.

#### Network Interception (All Vectors)
```
- fetch() - Hooked with behavioral analysis
- XMLHttpRequest - Open/send intercepted
- navigator.sendBeacon - Critical for exit tracking (adds EXIT_TRACKING signal)
- Image.prototype.src - Catches tracking pixels (1x1 gifs, Facebook/LinkedIn pixels)
```

#### Cookie Interception
```javascript
// Intercepts document.cookie setter
// Detects tracking cookies by:
// - Known prefixes: _ga, _gid, s_vi, AMCV_, _fbp, _mkto_trk, __hstc, _hjid, _6si, etc.
// - Long expiry (1yr+) = PERSISTENT_ID signal
// - UUID-like values = "Contains unique identifier" signal
```

#### Canvas Fingerprinting Detection
```javascript
// Hooks: toDataURL(), getImageData(), fillText()
// Detection criteria:
// - Suspicious canvas size (small, specific dimensions like 300x150)
// - Text rendered to canvas (fillText called)
// - Canvas not visible (display:none, offsetParent null)
// - Multiple text renders (pangram technique)
```

#### Font Fingerprinting Detection
```javascript
// Hooks: measureText()
// Detects font enumeration: >20 fonts tested + >50 measureText calls
// Emits FINGERPRINT_DETECTED with fontsTested count
```

#### Behavioral Signal Weights
| Signal | Weight | Detection Method |
|--------|--------|------------------|
| PERSISTENT_ID | +20 | 1yr+ cookie with identifier |
| SESSION_TRACKING | +15 | Session ID across pages |
| INTERACTION_DATA | +25 | mouse/click/scroll in payload |
| DEVICE_FINGERPRINT | +30 | screen/resolution/timezone/webgl in payload |
| EXIT_TRACKING | +20 | sendBeacon on unload |
| CONVERSION_DATA | +20 | revenue/transaction/order in payload |
| HIGH_RES_TIMING | +15 | 13+ digit timestamps |
| CROSS_DOMAIN | +25 | Request to different hostname |
| CANVAS_READ | +30 | Canvas fingerprinting detected |
| FONT_PROBE | +25 | Font enumeration detected |
| FORM_CAPTURE | +25 | form/input/submit in payload |
| PII_COLLECTION | +30 | email/phone/address in payload |

**Threshold:** `confidence >= 20` marks request as tracking

#### Vendor Guessing from Behavior (`guessVendorFromBehavior()`)

Scores vendors 0-100 based on:
- URL patterns (domain matches)
- Cookie presence (vendor-specific cookies)
- Payload signatures (vendor-specific parameters)

**30+ vendors supported:**
- Adobe: `/b/ss/`, `adobedc.net`, `demdex.net`, `s_vi` cookie, `mid|mcid|aamb` in payload
- Google: `google-analytics.com`, `/g/collect`, `_ga` cookie, `&tid=G-|UA-`
- 6sense: `6sense.com`, `6sc.co`, `_6si` cookie, `company_name|firmographic`
- Bombora: `bombora.com`, `ml314.com`, `"intent"|"topic"`
- Segment: `segment.io`, `anonymousId`, `"type":"track|page|identify"`
- Mixpanel: `mixpanel.com`, `device_id|distinct_id|\$properties`
- Facebook: `facebook.com/tr`, `fbp|fbclid`
- LinkedIn: `linkedin.com/px`, `snap.licdn.com`
- HubSpot: `hubspot.com`, `hs-scripts.com`, `__hstc` cookie
- Hotjar: `hotjar.com`, `_hj` cookies
- FullStory, Amplitude, Heap, Clearbit, Drift, Intercom, consent platforms...

#### First-Party Proxy Detection
```javascript
// Suspicious subdomains: metrics, analytics, tracking, events, telemetry,
//                        stats, data, insights, pixel, beacon, collect
// Suspicious paths: /analytics, /events, /tracking, /metrics, /collect,
//                   /beacon, /px, /pixel, /t/, /e/, /v1/track, /v1/page

// Deep inspection: If 2+ analytics patterns in payload, confidence 60-100%
// Patterns: event|page_view, user_id|anonymous_id, timestamp|ts, properties|context
```

**GTM/Tealium Awareness:**
```javascript
// When tag manager detected:
tagManagersDetected.add(vendorKey);
scheduleRescan(); // Rescans at 500ms, 1500ms, 3000ms, 5000ms

// MutationObserver catches dynamically injected scripts
// If tag manager active + new script found -> rescan in 100ms
```

**Inline Script Detection:**
- GTM container IDs: `GTM-[A-Z0-9]+`
- GA4 measurement IDs: `G-[A-Z0-9]+`
- UA properties: `UA-\d+-\d+`
- Tealium: `utag` or `tealium` in content

- Also checks iframes for vendor domains

## Architecture Overview

```

│                         BROWSER PAGE                             │
├─────────────────────────────────────────────────────────────────┤
│  hook.js (PAGE CONTEXT)                                         │
│  ├── Intercepts: fetch, XHR, beacon, Image.src, document.cookie │
│  ├── Detects: canvas/font fingerprinting                        │
│  ├── Analyzes: behavioral signals (12 types)                    │
│  ├── Guesses: vendor from behavior (30+ vendors)                │
│  └── Emits: postMessage to bridge.js                            │
├─────────────────────────────────────────────────────────────────┤
│  detector.js (CONTENT SCRIPT - ISOLATED)                        │
│  ├── Scans: <script src>, inline scripts, iframes               │
│  ├── Watches: MutationObserver for dynamic injection            │
│  ├── Tracks: tag managers (GTM, Tealium, Ensighten)             │
│  └── Sends: chrome.runtime.sendMessage                          │
├─────────────────────────────────────────────────────────────────┤
│  bridge.js (CONTENT SCRIPT - ISOLATED)                          │
│  └── Relays: postMessage ↔ chrome.runtime.sendMessage           │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│  background.js (SERVICE WORKER)                                  │
│  └── Routes messages to side panel                               │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│  sidepanel/main.js (SIDE PANEL UI)                              │
│  ├── Receives: all message types                                 │
│  ├── State: vendors, identities, network, cookies, fingerprints │
│  ├── Renders: expandable vendor cards with behavioral evidence  │
```
