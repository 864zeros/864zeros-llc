# Audio Files Required

This extension requires 4 seamless loop audio files. Place them in this directory:

| File | Type | Description |
|------|------|-------------|
| `white_noise.mp3` | White Noise | All frequencies, consistent static |
| `pink_noise.mp3` | Pink Noise | Softer, low-frequency emphasis |
| `brown_noise.mp3` | Brown Noise | Deep rumble, heavy low-frequency |
| `gentle_rain.mp3` | Gentle Rain | Natural ambient, predictable pattern |

## Requirements

- Format: MP3 (high quality, 128kbps minimum)
- Duration: 30-60 seconds (will loop seamlessly)
- Seamless loop: Audio must loop without audible gaps or clicks
- Royalty-free: Must be licensed for commercial use

## Recommended Sources

- Freesound.org (filter by Creative Commons licenses)
- Pixabay (free for commercial use)
- Zapsplat (free with attribution)

## Testing

After adding audio files, load the extension and verify:
1. Each sound plays without errors
2. Loops are seamless (no audible seam on repeat)
3. Audio continues when panel closes
