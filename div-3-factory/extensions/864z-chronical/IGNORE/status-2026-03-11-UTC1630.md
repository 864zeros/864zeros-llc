## Session Summary

Implemented several enhancements following clipboard extension patterns:

### 1. Download Markdown (Full Conversation)
- Added download button to each entry card

- Header includes: source LLM, date, time, message count
- Filename format: `chronicle-{scribe}-{title}-{date}.md`

### 2. Entry Card Redesign (Clipboard Pattern)
Refactored entry cards to match clipboard's `clip-card` pattern:

```
.entry-card
├── .entry-title
├── .entry-excerpt
├── .entry-card__meta
│   ├── scribe badge (colored per platform)
│   ├── URL link (domain, opens original)
│   ├── time ago
│   └── message count
└── .entry-card__actions
    ├── star (toggle)
    ├── download (markdown)
    └── delete
```

### 4. Chat URL on Card
