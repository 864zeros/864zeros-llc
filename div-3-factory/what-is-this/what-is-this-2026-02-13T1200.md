## Core Principles (Non-Negotiable)

1. **KISS** — Ship the smallest working version. No over-engineering.
2. **Privacy First** — All data stored locally. No analytics, no telemetry, no tracking. PPI redacted before any AI call.
3. **No Ads. Ever.** — Revenue from one-time purchases and optional tipping only.
4. **ADHD-Friendly UX** — One primary action per screen, no streaks/guilt, large touch targets, instant feedback.
5. **Modular Architecture** — Shared code in `/lib/`, reusable across products.
